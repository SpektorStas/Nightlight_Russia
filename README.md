# Средняя ночная освещенность российских населенных пунктов в 2012-2023 гг.
Средняя освещенность российских населенных пунктов с 2012 по 2023 года
Используются ночные спутниковые снимки [VIIRS](https://eogdata.mines.edu/products/vnl/), средняя годовая освещенность, с 2012 по 2023 годы. Для 2012-2021 гг. берется версия 21, а для 2022-2023 гг. – версия 22. Данные по населенным пунктам взяты из [ИНИД](https://data.rcsi.science/data-catalog/datasets/160/)

## Предобработка ночных спутниковых снимков
Используемые ночные спутниковые снимки перед анализом предобрабатываются. Значениям пикселей ниже нуля присваиваются значения 0 (так как отрицательной светимости не может быть), а значения выше 650 исключаются (аномалии в данных, шумы, свет от теплиц).

## Обработка данных
Для обработки были взяты населенные пункты с населением более 10 человек, всего 95 тыс. населенных пунктов. Вокруг каждого из них строится буфер радиуса 1 км для всех населенных пунктов с населением менее 1000 человек, радиуса 5 км для населенных пунктов с населением 1-10 тыс. человек, радиуса 10 км для населенных пунктов с населением 10-100 тыс. человек, радиуса 20 км для населенных пунктов с населением 100 тыс. - 1 млн человек, и 40 км для городов-миллионников. 

Далее считаются статистики ночных спутниковых снимков в этих буферах: среднее, медиана, стандартное отклонение, максимум, мода, вариация, дисперсия. В репозиторий загружены первые 100 строк, а полную версию можно взять из [яндекс диска](https://disk.yandex.ru/d/smX54MvfifWSww).

## Анализ и верификация
Полученные данные согласовываются с интуицией и предположениями. Во-первых, наибольшие уровни освещенности в крупных городах. Разницы между городами миллионниками и городами с населением 500 тыс. - 1 млн чел. почти нет. Это может объясняться тем, что города миллионники имеют больший радиус, в пределах которого считаются статистики, из-за чего в него попадают неосвещенные территории, удаленные от центра, в то время как города с населением 500 тыс. - 1 млн чел. еще не так сильно разрослись и включают в себя меньше неосвещенных территорий.
